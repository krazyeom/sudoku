# Sudoku Studio

Modern Sudoku app built with Next.js.

## Run locally

```bash
npm install
npm run dev
```

The app runs on **http://localhost:6767**.

## PM2

```bash
bash scripts/start.sh
bash scripts/stop.sh
```

## Notes

- Difficulty presets: **하 / 중 / 상**
- Generated puzzles are validated for uniqueness before they are shown
- Use the keypad or your keyboard (`1-9`, `Backspace`, `Delete`)
